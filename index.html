UNIT I - Introduction to C++

C++ is a general-purpose programming language created by Bjarne Stroustrup in 1985 as an extension of the C language. It was designed to add object-oriented features to C while maintaining C's efficiency and flexibility.

Key Features of C++

  1. Object-Oriented Programming: C++ supports OOP concepts like classes, objects, inheritance, polymorphism, encapsulation, and abstraction. This helps model real-world problems using objects.
  2. Middle-Level Language: C++ combines features of both high-level and low-level languages. You can do system programming like C and also build large applications with OOP.
  3. Compiled Language: C++ code is compiled into machine code, which makes programs run very fast compared to interpreted languages.
  4. Rich Library Support: C++ provides Standard Template Library (STL) which includes built-in data structures and algorithms like vectors, stacks, queues, sorting, etc.
  5. Platform Dependent: C++ programs need to be recompiled for different operating systems, unlike Java which is platform independent.

Why C++ was Introduced

C language had limitations for large software development:

  1. No data security because of global variables
  2. No code reusability
  3. Difficult to maintain large codebases

C++ solved these by introducing classes and objects, which bind data and functions together. This is called encapsulation.

Basic Structure of C++ Program

#include <iostream>  // header file for input/output
using namespace std; // allows use of standard names

int main() {         // main function, execution starts here
    cout << "Hello"; // output statement
    return 0;        // indicates successful termination
}

I/O in C++ - Input and Output

1. Output - cout

#include <iostream>
using namespace std;
int main() {
    cout << "Hello World";
    return 0;
}

2. Input - cin

#include <iostream>
using namespace std;
int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;
    return 0;
}

Decision Making Statements

int marks = 85;
if (marks >= 90) {
    cout << "Grade A";
} else {
    cout << "Grade B";
}
